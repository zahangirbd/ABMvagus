Files and Folder Information
============================
1. Additional file 2.zip:
	It contains ABM executable with standard landscape file and weather profile as input
	Follow the instructions written in ReadMe.txt file inside of the zip file. 

2. Additional file 3.xml: 
	It is a landscape file which can be used as input for ABM.
	To set this as input file then replace "StandardLandscape.xml" by "Additional file 3.xml" int the run.bat/run.sh file of Additional file 2.

3. Additional file 4.csv:
	It is weather profile data which is used as input for ABM.
	To set this as input file then replace "WeatherData.csv" by "Additional file 4.csv" int the run.bat/run.sh file of Additional file 2.
